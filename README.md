# Breast Cancer Diagnosis Framework 🧬
**Classification of Malignant vs. Benign Tumors**

This repository contains a comprehensive Machine Learning framework developed for the **ITGN 430 – Artificial Intelligence** course.

## 📋 Project Objective
The goal is to analyze numerical features derived from biopsy images to classify tumors as **Malignant (M)** or **Benign (B)**.

## 🛠️ The Framework Pipeline
1. **Data Exploration:** Statistical analysis of 30 numerical features.
2. **Preprocessing:** Label Encoding and Feature Scaling (StandardScaler).
3. **Model Selection:** Comparison of Random Forest, SVM, MLP, and KNN.
4. **Evaluation:** Scoring based on Accuracy, Precision, Recall, and AUC-ROC.
